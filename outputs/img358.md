{ राम को रूप निहारति जानकी कैंगन के-नग की परदाही। } **शृंगार रस**  
 { याते सबे सुख झूले गइ कर तेकि रही पल तारति नाही ॥ } (रामे)

{ जिनेहि दिसि नारद बँढे फूली। सो दिसि तेहि न बिलेकी झूली। } **हास्य रस**  
 { पुनि-पुनि मुनि ठकसहिं झकुलाही। दोखि दसा हर-गन भुसकाही ॥ } (हास्य)  
 [नारद के वानर रूप की कल्पना करके आक्षेप किया गया है।]

{ पांय बेहाल विषाइन सो अये, कंठक-जाल लैगे पुनि जोये। } **करुण रस**  
 { डाय! मढादुःख पाये सखा ! तुम डायें इतें न किंतें दिन खोये ॥ } (शोक)  
 (सुदामा की दयनीय दशा पर टूटने की चिन्तन)

{ जय के दृढ़-विश्वास-युक्त धे दीपीमान जिनेके मुख-भेंडल। } **वीर रस**  
 { पर्वत की भी खण्ड-खण्ड कर रजकण कर देवे भी चंचल ॥ } (इत्साह)

{ माखे लपन, कुटिल भयी भौंहै। } **रौद्र रस**  
 { रद-पट फरकत नैन रिस्सौंहै ॥ } (क्रोध)  
 { कहि न सकत रघुवीर डर, लगे कचन जनु खान। }  
 { नाइ राम-पद-कमल-भुग, बोले गिरा प्रसाद ॥ }
