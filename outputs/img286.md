## विसर्ग संधि

### (i) दीर्घ सन्धि:

लघु या दीर्घ **अ, इ, उ, ऋ** के बाद लघु या दीर्घ **अ, इ, उ, ऋ** आने पर दोनों के स्थान पर दीर्घ स्वर हो जाता है।

- अ + अ = आ → देव + अधिपति = देवाधिपति
- अ + आ = आ → हिम + आलय = हिमालय
- आ + अ = आा → युवा + अवस्था = युवावस्था
- आ + आा = आा → महा + आशय = महाशय
- इ + ई = ई → कवि + ईश = कवीश
- इ + ई = ई → अभि + इष्ट = अमीष्ट
- ई + ई = ई → जानकी + ईश = जानकीश
- ई + ई = ई → लक्ष्मी + इच्छा = लक्ष्मीच्छा
- उ + ऊ = ऊ → सिन्धु + ऊर्षि = सिन्धुर्षि
- ऊ + उ = ऊ → वधू + ऊर्षि = वधूर्षि
- ऋ + ऋ = ऋ → पितृ + ऋण = पितृण

### (ii) यण सन्धि:

लघु या दीर्घ **इ, उ, ऋ** के उपरंत कोई असमान स्वर आये तो **ई, उ, ऋ** के स्थान पर क्रमशः **य, र, व** हो जायेगा

- इ + अ = य → यदि + अपि = यदापि
- ई + अ = य → देवी + अर्पण = देव्यर्पण
- इ + आा = या → अति + आचार = अत्याचार
- ई + आा = या → देवी + आलय = देव्यालय
- उ + उ = यु → पति + उपकार = प्रत्युपकार
- ऊ + ऊ = यू → पति + ऊष = प्रत्युष
- ई + ऊ = यू → वाणी + ऊर्षि = वाण्युर्षि
- ऊ + ई = वी → वधू + ईश्वरी = वध्वीश्वरी

- उ + अ = व → अनु + अय = अन्वय
- ऊ + अ = व → वधू + अर्ध = वधर्ध
- उ + आा = वा → मधु + आलय = मधवालय
- ऊ + आा = वा → वधू + आगमन = वधवागमन

- ऋ + अ = र → मातृ + अर्ध = मातार्ध
- ऋ + आा = रा → पितृ + आरा = पितूरा
