③ अयादि सन्धि: जब ए, ऐ, ओ, औ के बाद कोई भी स्वर आये तो,  
 ↓ ↓ ↓ ↓  
 अयु आयु अव आव

ए + अ = अयु → ने + ङन = नयन  
 ऐ + अ = आयु → नै + अक = नायक  
 ओ + अ = अव → ओ + अन = अवन  
 औ + अ = आव → औ + अक = पावक

④ वृद्धि सन्धि: जब लघु या दीर्घ 'अ' के बाद ए, ऐ, ओ, औ आए तो  
 ए → ऐ, ऐ → ओ, औ → औ हो जाता है।

अ + ए = ऐ → परम + ऐश्वर्य = परमैश्वर्य  
 आ + ए = ऐ → तथा + एव = तथैव  
 आ + ऐ = ऐ → महा + ऐश्वर्य = महैश्वर्य  
 अ + ऐ = ऐ → धर्म + ऐक्य = धर्मैक्य  
 अ + ओ = औ → जल + ओक = जलौक  
 अ + औ = औ → जल + औषधि = जलौषधि  
 आ + ओ = औ → महा + ओज = महौज  
 आ + औ = औ → महा + औदार्य = महौदार्य

⑤ गुण सन्धि: 'अ' या 'आ' के बाद लघु या दीर्घ इ, उ, ऋट आये तो दोनों के स्थान पर क्रमशः (ए, औ) (अरु) हो जाता है।

अ + इ = ए → देव + इन्द्र = देवेन्द्र  
 आ + इ = ए → महा + इन्द्र = महैन्द्र  
 अ + ई = ए → परम + ईश्वर = परमेश्वर  
 आ + ई = ए → महा + ईश्वर = महैश्वर  
 अ + उ = ओ → पर + उपकार = परोपकार  
 आ + उ = ओ → महा + उत्सव = महोत्सव  
 आ + ऊ = ओ → गंगा + ऊर्मी = गंगोर्मी  
 उ + ऋट = अरु → देव + ऋटि = देवर्षि  
 आ + ऋट = अरु → राजा + ऋटि = राजर्षि

⑥ पूर्व रूप सन्धि: 'अ', 'ए' या ओ के बाद आये तो 'अ' के स्थान पर पूर्व रूप का चिन्ह "s" हो जाता है।  
 लोको + अयम = लोकोसयम, हरे + अव = हरेऽव  
 हरे + अल = हरेऽल, प्रभो + अवतृ = प्रभोऽवतृ
